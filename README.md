# Gee - 从零实现的 Go Web 框架
Gee 是一个**学习性质**的 Go Web 框架，通过 7 天迭代从零构建，深入理解 Web 框架的核心原理。API 设计参考 Gin 框架。

> 📖 **配套教程**：[7天用Go从零实现Web框架Gee](https://geektutu.com/post/gee.html)

## 🚀 快速体验


```bash
# 克隆项目
git clone https://github.com/baeonx/go-web-gee.git

# 进入对应天的示例目录
cd go-web-gee/day1

# 运行示例
go run main.go
```

## 📅 开发路线图
Day 1 - 框架雏形 & 路由映射
Day 2 - 上下文 Context & 路由分离
Day 3 - 动态路由 (Trie 树)
Day 4 - 路由分组控制
Day 5 - 中间件链
Day 6 - 模板引擎支持
Day 7 - 错误恢复机制
